Seja bem-vindo a documentação do Wide Pay! Aqui você encontrará as informações necessárias para fazer a integração com nossa infraestrutura.

Visão geral

Antes de iniciar, crie sua conta no Wide Pay.

Com a API do Wide Pay é possível utilizar qualquer recurso disponível em sua conta, como gerar transações, carnês, assinaturas, receber notificações toda vez que algum status sofrer alteração.

Também é possivel utilizar os recursos de pagamento de contas, recargas de celular, transferências bancárias, transferências entre contas Wide Pay e transferências para cartões pré-pagos.

Carteira e token

Para utilização da API é necessário ter o ID da carteira e o token de autenticação.

O conceito de carteiras do Wide Pay serve para que você possa separar os recebimentos, pagamentos e transferências. Caso de você tenhha duas lojas por exemplo.

Para ter um gerenciamento completo de suas carteiras como criar, editar, obter ID e token, logue em sua conta Wide Pay, acesse o menu superior Configurações, depois clique em Carteiras.

Ambiente de testes

A configuração do ambiente de teste é feita na hora que você adiciona a carteira no Wide Pay.

Após feito os testes de integração, você poderá migrar a carteira para o ambiente de produção. Lembramos que uma carteira já em produção não pode ser migrada para o ambiente de testes.

Endpoint e respostas

Todas as respostas da API são em JSON e as requisições são feitas no endpoint:

https://api.widepay.com

Exemplos

Nos exemplos que você encontrará pela documentação, usaremos o cURL e os códigos compatíveis com as bibliotecas disponíveis.

Dúvidas ou sugestões

Mande um e-mail para contato@widepay.com em caso de dúvidas ou sugestões de melhorias e integrações da API.
